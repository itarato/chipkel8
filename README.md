Chip8 Emulator
--------------

![Screenshot](./misc/screenshot.png)

## Setup

- https://www.haskell.org/ghcup/
- `sudo apt install freeglut3-dev`

## Use

```bash
cabal run exes -- <ROM-PATH>
```

## Test

```bash
cabal test
```
